# AcessívelBR Hub 🚀
**Projeto Desenvolvido para o Grand Prix SENAI de Inovação - Desafio Petrobras**

## 🎯 O Desafio
A Petrobras identificou que, embora existam diversas iniciativas de acessibilidade (RH, Saúde, Ergonomia, TI), elas ocorrem de forma **descentralizada e isolada (silos)**. Isso resulta em:
*   Processos lentos, manuais e reativos.
*   Falta de integração entre as soluções das diferentes áreas.
*   Baixa visibilidade de problemas recorrentes e causas raiz.

## 💡 Nossa Solução: O Hub de Centralização Inteligente
O **AcessívelBR Hub** não tenta substituir os programas que já funcionam na Petrobras; ele atua como o **conector central** de todos eles, orchestrando as iniciativas existentes através de Inteligência Artificial.

### Diferenciais Técnicos:
*   **Centralização Real:** Unifica os dados de RH, TI, Saúde e Engenharia em uma plataforma única de comando.
*   **Triagem e Roteamento via IA:** Lê o contexto das demandas e as direciona automaticamente para a área responsável em segundos, eliminando o "ping-pong" burocrático.
*   **Clustering (Detecção de Padrões):** Nossa IA identifica quando um mesmo problema se repete em diferentes unidades (ex: prédios ou refinarias), permitindo que a empresa resolva a causa raiz de uma vez só.
*   **IA de Apoio à Decisão (Human-in-the-loop):** A tecnologia não substitui o humano; ela fornece recomendações e "dicas" baseadas em dados para facilitar a decisão final do gestor.
*   **Mapa de Calor Estratégico:** Visibilidade geográfica em tempo real de todos os gargalos de acessibilidade da companhia através de um Dashboard intuitivo.

## 🛠️ Tecnologias Utilizadas
*   **Next.js 16 (Turbopack)** + React 19
*   **TypeScript** (Garantia de robustez e segurança do código)
*   **Tailwind CSS** (Interface moderna, limpa e responsiva)
*   **Lucide React & Base UI** (Componentes focados em acessibilidade)
*   **Framer Motion** (Animações fluidas para melhor UX)

## 🚀 Como Rodar o Projeto
1.  Clone o repositório.
2.  Instale as dependências (utilizando a flag de peer-deps para compatibilidade com a biblioteca de mapas):
    ```bash
    npm install --legacy-peer-deps
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
    *O projeto está configurado para rodar na porta 7777.*
